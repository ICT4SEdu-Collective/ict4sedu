# ICT4S Education Repository

This repository is a collection of resources for teaching and learning about sustainability in computing. It is part of the [ICT4S Education Initiative](https://ict4sedu.wordpress.com), which aims to support educators and students in integrating sustainability into computing education.

# 👩🏽‍🏫 Courses

Our [reusable teaching artfacts repository](Courses.md) follows a generic structure with a descriptive card per course and folders to browse and offer the materials.
Instructions on this will follow soon along with a template.

# 📚 Reading Lists

A [list of our favorite essentials](favourite-readings.md) to get started with sustainability education if coming from IT.

# 👨🏻‍⚖️ Licences

When sharing documents, courses, or educational materials, it's important to choose the right license. Therefore, we maintain a list of 
[common open source licenses](licenses.md) that can be used for educational content, along with tips on how to choose the best one for your needs.
